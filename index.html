Welcome to the Pomodoro Timer Game.
This Game was designed to bring gaming and productivity together in one application.
The idea is to get as much tasks as possible while working in certain timed sessions.
Here's the rules for the game below:

  • 1. Add All Tasks to The To Section
  • 2. Add Priority Markers on All Task. Reds Get 500pts, Yellow Get 300pts and Green gets 100pts
  • 3. Set Your Timer for a Short or Long Session
  • 4. Get As Much as Task Done as Possible During Your Work Session to Accumulate Points
